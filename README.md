# Customer_Churn_Analysis
Retail sales analytics project using Python, PostgreSQL, SQL, and Power BI to clean transaction data, analyze customer behavior, track revenue trends, and visualize product and country performance.


# Online Retail Customer Behavior & Sales Analysis

## Overview
Analyzed online retail transaction data to uncover customer behavior, sales trends, product performance, and regional insights using Python, PostgreSQL, SQL, and Power BI.

## Tools
- Python
- Pandas
- PostgreSQL
- SQL
- Power BI

## Workflow
1. Cleaned and processed raw retail data using Python.
2. Performed SQL analysis for customer, product, and sales insights.
3. Built an interactive Power BI dashboard with KPIs and visualizations.

## Key Analysis
- Top customers by revenue
- Repeat vs one-time customers
- Top-selling products
- Country-wise sales performance
- Monthly revenue trends

## Dashboard Features
- KPI cards
- Sales trend charts
- Product analysis
- Country performance visuals
- Customer behavior insights
